# 🔐 LookerRepo

LookerRepo is a lightweight Python cryptographic toolkit for secure password-based encryption and decryption.

## ✨ Features

- AES-256-CBC encryption
- PBKDF2 key derivation (SHA-256)
- Secure random salt generation
- Base64-safe output
- CLI interface
- Clean and readable implementation
